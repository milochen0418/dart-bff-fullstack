# Delta Update
This example give the Delta Update pattern for dart web server and flutter client through websocket. 

# Run for server
## run server with ask y/N
./serverrun.sh  
## run server without ask y/N
yes | ./serverrun.sh  

# Run for client 
## show running devices
./uirun.sh --list    
## run for the specific device
./uirun.sh -d xxxxdev
